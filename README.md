## üöÄ Getting Started

1. **Install dependencies**
```sh
bun i
```  

or 

```sh
npm i
```

(replace bun with npm from now on if you don't have bun)

2. **Init environment**
```sh
bun run init
code .env
```

Replace variables with appropriate values. `ADMINS` is a comma-seperated list.

3. **Run the bot**
```sh
bun run start
```
